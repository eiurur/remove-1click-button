remove-1click-button
====

Amazonの「1Clickで今すぐ買う」ボタンを削除し、「今すぐ読む」ボタンに置き換えます。

**NOTE**

Kindle Unlimitedに対応していない本の場合、サンプルページに飛びます。

「Kindle Unlimited マイブック」に登録中の本であればそのまま閲覧ページに飛びます。未登録であればサンプルページに飛びます。
